# 🎯 손안에 가이드 사용자 튜토리얼

**손안에 가이드**는 여행 중 사진과 음성으로 언제든지 풍부한 정보를 얻을 수 있는 AI 기반 모바일 가이드입니다.

---

## 📱 앱의 특징

### 🤖 Gemini AI 기반 콘텐츠
- **사진 1장만으로** 문화유산, 음식, 건축물, 예술작품에 대한 풍부한 해설 자동 생성
- **음성 질문**으로 복잡한 검색 없이 필요한 정보만 추출
- 약 **10초 이내**로 음성+텍스트 결과 제공

### 📍 위치 기반 저장
- GPS 데이터 자동 저장으로 나중에 **"이건 어디였더라?"** 해결
- 여행 중 촬영한 모든 가이드는 지도에 표시 가능

### 💾 오프라인 완벽 지원
- 한 번 생성된 가이드는 **네트워크 없이도 즉시 재생**
- 비행기 탈 때도, 산 중턱에서도 언제든지 음성 해설 청취 가능

### 🎁 추천 리워드 & 캐시백
- 친구에게 공유한 링크로 신규 가입 시 → **쌍방 각 +10 크레딧**
- 친구가 충전할 때마다 → **추천인 +20 크레딧 보너스**
- 1,000 크레딧 이상 모으면 → **현금 환급 신청 가능** (카카오페이/계좌)

## 🎮 메인 화면 버튼 가이드

### 하단 4개 버튼 (항상 표시됨)

```
📷 촬영        🎙️ 음성        📸 업로드        📁 보관함
```

#### 📷 **카메라 버튼** (왼쪽)
**기능**: 실시간으로 사진 촬영  
**사용법**:
1. 버튼 누르기
2. 카메라가 켜지면 피사체에 맞춤
3. 버튼 재클릭으로 촬영
4. AI가 자동으로 분석 (로딩 약 10초)

**팁**: 건물, 음식, 예술작품 등 명확한 피사체가 있을수록 정확한 설명 생성

#### 🎙️ **음성 버튼** (왼쪽에서 두 번째)
**기능**: 마이크로 음성 질문  
**사용법**:
1. 버튼 누르고 계속 누르기 (청색 파동이 나타나면 녹음 중)
2. 질문 음성 입력 예:
   - "이건 뭐예요?"
   - "파리의 유명한 까페는?"
   - "이 건축물의 역사가 뭐죠?"
3. 마이크 버튼 놓기
4. AI가 음성을 이해하고 답변 생성

**팁**: 명확하고 느리게 말할수록 인식률 높음

#### 📸 **업로드 버튼** (오른쪽에서 두 번째)
**기능**: 휴대폰 갤러리에서 사진 선택  
**사용법**:
1. 버튼 누르기
2. 갤러리 열림
3. 여행 중 촬영한 사진 또는 아무 사진 선택
4. AI 분석 시작

**팁**: 여러 사진 선택 가능 (한 번에 여러 개 생성 가능)

#### 📁 **보관함 버튼** (오른쪽)
**기능**: 저장된 모든 가이드 보기  
**사용법**:
1. 버튼 누르기
2. 3×3 그리드로 모든 가이드 사진 표시
3. 사진 클릭 → 상세 정보 확인
4. 여러 개 선택 → 공유 가능

**상세 페이지 버튼**:
- 🔊 **음성 재생** → 생성된 해설을 음성으로 청취
- 📝 **텍스트 보기** → 전문 텍스트 읽기
- 💾 **저장** → 보관함에 추가 (이미 저장된 항목도 재저장 가능)

## 📖 상세 정보 화면 가이드

### 자동 표시되는 정보
- **배경 이미지**: 촬영/업로드한 사진 (투명 오버레이)
- **위치 정보**: GPS로 자동 추출한 장소명 (하얀 박스)
- **AI 해설**: Gemini 생성 텍스트

### 하단 3개 버튼

```
🔊 음성 재생        📝 해설 읽기        💾 저장하기
```

#### 🔊 **음성 재생 버튼**
- **Press**: 음성 해설 자동 시작 (첫 문장부터)
- **특징**: 문장마다 파란 하이라이트 표시 + 자동 스크롤
- **재생 중**: 버튼이 ⏸️ (일시정지)로 변함
- **다시 누르기**: 일시정지/재개
- **Windows만 지원**: Microsoft Heami 음성 (Mac에서는 시스템 음성)

**팁**: 음성 + 텍스트 동시 읽기로 이해도 향상

#### 📝 **해설 읽기 버튼**
- **기능**: 전체 텍스트 또는 요약본 표시 토글
- **텍스트 보기**: 스크롤하면서 전체 내용 읽기 가능
- **음성과 함께**: 음성이 재생되면 현재 문장이 하이라이트됨

#### 💾 **저장하기 버튼**
- **기능**: 현재 가이드를 보관함에 저장
- **저장 후**: 토스트 메시지 "저장되었습니다"
- **재저장**: 이미 저장된 항목도 언제든 다시 저장 가능

## 🔗 공유 기능 가이드

### 공유 페이지 생성

#### 방법 1: 보관함에서 직접 선택
1. 📁 **보관함** 버튼 클릭
2. 공유하고 싶은 가이드 **3개~30개 클릭** (체크박스 선택)
3. **"공유"** 버튼 클릭
4. 공유 페이지 생성 완료!

#### 방법 2: 공유 페이지 커스터마이징
1. 공유 링크 생성 시:
   - **페이지 이름**: "파리 맛집 투어"
   - **보내는 사람**: "민준이"
   - **장소명**: "파리"
2. 확인 버튼 → **단축 URL 생성** (예: `/s/abc12345`)
3. 링크 복사 → SNS 공유

### 공유 페이지 특징

```
┌─────────────────────────────┐
│  📸 3단 응답형 갤러리 레이아웃 │
│  ┌─────────────────┬──────┐ │
│  │                 │      │ │
│  │  이미지 1       │ 이미지2│
│  │                 │      │ │
│  ├─────────────────┼──────┤ │
│  │  이미지 3 (큰)   │      │ │
│  └─────────────────┴──────┘ │
│                             │
│  🎵 Microsoft Heami 음성    │
│  (Windows 공유 페이지 내 자동 재생) │
└─────────────────────────────┘
```

- **반응형 디자인**: 모바일, 태블릿, PC 모두 최적화
- **음성 자동 재생** (Windows Chrome에서만)
- **추천 코드 자동 주입**: 친구가 링크 접속 → 쿠키 저장 → 가입 시 추천인 연결

## 💰 크레딧 시스템 이해하기

### 크레딧은 무엇?
- **AI 호출마다 소비** (가이드 1개 생성 = 약 1 크레딧)
- **Stripe로 구매**: 10 EUR = 140 크레딧
- **추천 보너스**: 자동 적립 (가입, 충전 시)

### 충전 프로세스
1. **프로필 페이지** 접속
2. **충전하기** 버튼
3. **Stripe 결제** (신용카드)
4. **자동 적립** (즉시 반영)

### 추천 리워드 상세
| 상황 | 나 | 추천인 |
|------|-----|--------|
| 신규가입 (추천링크 사용) | +10 | +10 |
| 나 충전 시 | +140 | +20 |
| 친구가 충전 | - | +20 |

### 캐시백 (현금 환급)
- **최소 신청액**: 1,000 크레딧
- **환율**: 1,000 크레딧 = 100 EUR (추정)
- **신청**: 프로필 → 캐시백 → 신청
- **지급**: 관리자 검토 후 5영업일 이내 (카카오페이/계좌이체)
- **상태**: pending (대기) → approved (승인) → rejected (거절)

## ⚙️ 기술 사양 & FAQ

### Q: 인터넷 없이도 쓸 수 있나요?
**A**: 네! 한 번 생성된 가이드는 **완전히 오프라인에서 재생** 가능합니다.
- GPS 좌표 저장됨
- 음성 파일 로컬 저장
- 비행기 모드에서도 작동

### Q: 사진 품질이 나쁘면 어떻게 되나요?
**A**: AI는 흐린 사진도 분석하지만, 명확한 사진일수록 정확합니다.
- 피사체를 프레임 중앙에 배치
- 밝은 곳에서 촬영
- 흔들리지 않게 촬영

### Q: 음성 인식이 안 돼요
**A**: 음성 관련 확인 사항:
1. 마이크 권한 허용 확인
2. 시끄러운 곳에서는 크리스탈 음성으로 천천히 말하기
3. "이게 뭐죠?" 같은 간단한 질문부터 시작

### Q: 저장된 가이드를 수정할 수 있나요?
**A**: 현재 버전에서는 **수정 불가**입니다.
- 새로 촬영 → 새 가이드 생성
- 삭제 후 재생성 권장

### Q: 크레딧 사용 패턴은?
**A**: 
- 사진 1장 (AI 분석) = ~1 크레딧
- 음성 질문 = ~1 크레딧
- 저장: **무료**
- 공유: **무료**
- 음성 재생: **무료** (생성 후 무제한)

### Q: 개인 데이터는 안전한가요?
**A**: 
- **Replit 인증** (Google OAuth) 사용
- PostgreSQL **암호화된 DB 저장**
- 사진은 임시 처리 후 삭제
- 개인정보 공개 안 함

## 💡 프로팁

### 최고의 결과를 위한 팁

1. **사진 품질**
   - 자연광에서 촬영
   - 피사체를 중앙에 배치
   - 너무 멀리서 찍지 않기

2. **음성 질문**
   - "이게 뭐예요?" → 분석
   - "왜 유명해요?" → 역사/배경
   - "먹을 수 있어요?" → 용도 설명

3. **오프라인 활용**
   - 여행 가기 전 미리 생성
   - 비행기 탈 때 다운로드 (자동 저장)
   - 산 중턱에서도 음성 청취

4. **크레딧 절약**
   - 추천 프로그램 최대 활용
   - 중요한 가이드만 생성
   - 한 번 생성한 가이드는 무제한 재생
